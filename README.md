# Excel 表转换与道具检索工具

## 功能概述

- 上传 zip/rar（内含多个 xlsx），批量处理并下载处理后的 zip
- 表文件重命名：`原表名(翻译后的中文).xlsx`
- 每个 sheet 结构改造：插入行/列、写入 A1~A3 标记、将第 1 行表头翻译为中文并写入第 3 行
- 名称搜索：支持“精准匹配 / 包含匹配”切换
- 表分类：按“原表名包含类型英文关键字”归类并标注类型中文名
- 同类表二次检索：按 `id` / `key` / `B 列`（修改后第 2 列）进行匹配

## 运行方式（本地）

1. 安装依赖

```bash
pip install -r requirements.txt
```

2. 启动工具

```bash
streamlit run app/main.py
```

3. 在页面中上传 zip 或 rar，填写类型与名称，点击“开始处理并搜索”，最后下载处理后的 zip。

## 重要约束

- 翻译必须走外部免费翻译服务，默认使用 MyMemory（HTTP 请求），不使用任何本地翻译库。
